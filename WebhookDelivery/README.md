# WebhookDelivery

A .NET 9 webhook delivery service used to practice reliability patterns under real-world constraints. The project evolves through iterations, starting with a monolithic baseline and progressively swapping in Azure services.

## Project layout

- `Documentation/decisions/` holds Architecture Decision Records for this service.
- `Notes/` contains first-hand notes on Azure services and .NET libraries after use.
- `Journal/` is the working log of what was built, what broke, and what was learned.
- `Infrastructure/` holds Bicep templates for cloud-deployed iterations.
